## Prerequisites

Install SWI Prolog
https://www.swi-prolog.org/download/stable


## List URL

Landing Page

```sh
http://localhost:8000
```

Add Store

```sh
http://localhost:8000/add_store
```

Specific Store by Id

```sh
http://localhost:8000/store?id={id}
```


## Database Location

look up on folder Documents/Prolog and you will find data_umkm.db
